<h1 align="center">
  Aerocord
</h1>

Aerocord is a Discord client written in C#.<sup>(tasty glass included)</sup>

Supports Windows 7 - Windows 11

# What's working
- Login with email and password
- DMs
- Servers
- Websockets
- Sending messages
- Image viewing
- Video viewing
- Markdown Support
- Mica, Mica Alt and Acryllic Support
- Statuses
